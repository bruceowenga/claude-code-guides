# Claude Code Guides

A personal knowledge base of homelab guides generated with [Claude Code](https://claude.ai/claude-code). Each guide documents a specific setup, configuration, or operational procedure for self-hosted infrastructure.

## About

This repo serves as a living reference for my homelab — covering Docker services, Linux administration, networking, monitoring, and more. Guides are generated through real interactions with Claude Code while solving actual problems in the lab.

## Structure

Guides are organized by topic:

```
claude-code-guides/
├── docker/          # Container setup, Compose files, networking
├── monitoring/      # Observability, alerting, dashboards
├── networking/      # Cloudflare Tunnels, DNS, Nginx/reverse proxies
├── linux/           # System administration, storage, performance
├── security/        # Hardening, SSH, access control
└── services/        # Per-service setup guides (Ghost, etc.)
```

## Infrastructure

- **OS:** Linux (Ubuntu/CentOS)
- **Workloads:** Docker containers
- **Networking:** Cloudflare Tunnels
- **Tools:** Docker Compose, Nginx, standard Linux utilities

## Usage

Browse by topic folder or use GitHub search to find a specific guide. Each guide is self-contained with context, commands, and explanations.

## Contributing
